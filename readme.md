# Three Game in Lua

Консольная игра "3 в ряд" на Lua.  
Цель — объединять 3 и более одинаковых кристаллов в ряд по горизонтали (вертикали пока нет).  

## Особенности

- Поле 10x10 с 6 типами кристаллов (A-F).  
- Кристаллы падают вниз после удаления линий.  
- Новые кристаллы появляются сверху.  
- Возможность перемешивания поля, если нет возможных ходов.  
- Архитектура с разделением:
  - `GameModel` — логика игры, хранит поле.
  - `View` — вывод в консоль с цветами.
  - `main.lua` — основной цикл игры.  
- Подготовлено для будущих спец-кристаллов и бонусов (Crystal как объект).

## Требования
-- установленный и добавленный в переменные среды Lua (сработает команда lua -v в терминале/консоли) 

## Запуск 
-- На Linux/MacOS введите из корня проекта в терминал ./run/run.sh
-- На Windows введите из корня проекта в консоль .\run\run.bat 

## Структура проекта
├── main.lua     # основной цикл
